**버전:** v2  ·  **제작일:** 2026-07-16  ·  **판형:** 90 × 54 mm (표준 명함, 가로) · 양면

기술(AI)과 사람(커뮤니케이션) 사이를 잇는 사람을 브랜딩한 개인 프로젝트용 명함입니다.

> **v2 핵심 변경:** 앞면 오른쪽 여백에 **손으로 그린 인라인 SVG "연결(Connection)" 네트워크 성좌(constellation)** 그래픽을 추가했습니다.
> 잉크 저투명 텍스처로 조용히 깔려 이름의 강조를 해치지 않으며, 소유자의 연구 분야(**네트워크 분석**)와 뒷면 QR 격자와도 시각적으로 라임을 이룹니다.

## 🖨 인쇄 방법 (양면/듀플렉스)

1. `명함_RYU_GEONWOO_2026-07-16_v2.html` 파일을 **크롬(Chrome)** 에서 엽니다.
2. 화면의 **「🖨 인쇄 / PDF로 저장」** 버튼 또는 `Ctrl/Cmd + P`.
3. 인쇄 대화상자:
   - **용지 크기:** 90 × 54 mm (사용자 지정) — PDF 저장 시 자동 적용
   - **여백(Margins):** 없음(None)
   - **배경 그래픽(Background graphics):** 켜기 — QR·색상·네트워크 그래픽이 정확히 인쇄됩니다.
   - **배율(Scale):** 100%(기본 / "실제 크기")
4. 출력은 **정확히 2페이지**: **1페이지 = 앞면**, **2페이지 = 뒷면**.
5. **양면 인쇄:** 프린터 양면(듀플렉스) 옵션에서 **짧은 가장자리(Short edge) 넘김** 권장.
6. 인쇄소 발주 시 상하좌우 **3mm 재단여백(bleed)** 추가 요청. 모든 텍스트·그래픽은 안전여백 ≥4mm 안쪽에 배치되어 있습니다.

> ✔ 검증: 헤드리스 크롬 인쇄 PDF가 **2페이지 · MediaBox 255.12 × 153.12 pt (= 90 × 54 mm)** 로 확인되었습니다.

## 🛠 기술 메모 (v1 대비 추가분)

- **네트워크 그래픽:** `.card-front` 안에 `position:absolute; inset:0` 로 깔린 **인라인 `<svg viewBox="0 0 90 54">`** (1 유저 유닛 = 1mm). 텍스트 그룹(`.fc-top`/`.fc-bottom`)에 `position:relative; z-index:1` 을 주어 그래픽은 항상 텍스트 **뒤**에 위치 → 가독성 무손실. 노드=`<circle>` (fill-opacity 0.48~0.72), 연결선=`<line>` (stroke-opacity 0.30~0.36, 균일). 전체를 `<g transform="translate(3.5,-5)">` 로 감싸 우상단으로 이동(좌표 개별 수정 없이 연결 보존).
- 그 외 v1 기술 스택 **그대로 유지:** React 18.3.1 UMD, **Babel @7.26.4 고정**(8.x는 인라인 JSX를 깨뜨려 백지), QR base64 data URI를 `text/babel` **밖**의 `window.QR_URI` 전역으로 hoist, `@page{size:90mm 54mm;margin:0}` + `print-color-adjust:exact` 듀플렉스, `transform:scale(2.6)` 화면 미리보기, 전 구간 `word-break:keep-all`, 인라인 SVG 파비콘.
- 미리보기 캡처용 `?face=front|back` 쿼리 지원(단일 면 렌더, 스크린샷 전용).
