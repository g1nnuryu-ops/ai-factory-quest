**이미 만들어 본 것 (= 기술 선택지를 제안할 때 실현 가능하다고 봐도 되는 것):**
- CDN 기반 단일 `index.html` React 앱 (`single-react-dev` 에이전트 패턴) — 빌드 도구 없음
- Node/Express + Supabase(Postgres) 서버+DB 앱 (가계부, 쇼핑몰)

**개인 맥락:** 기혼, 3~7세 아이 2명. "아이와 함께 쓰는 앱"은 억지 설정이 아니라 실제로 유효한 타겟 사용자 옵션이므로, 개인용 앱을 정의할 때 선택지로 제시할 만하다.
