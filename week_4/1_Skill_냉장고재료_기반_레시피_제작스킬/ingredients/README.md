# 식재료 데이터 (ingredients)

3~7세 아이 2명을 둔 4인 가족이 **보통 보유하고 있는** 식재료를 재료당 1개의 JSON 파일로 정리한 데이터셋입니다.
`식재료 기반 레시피 제작 스킬`이 이 폴더를 읽어 보유 재료 기반으로 레시피를 추천/생성하는 데 사용합니다.

## 파일 구조

- 파일명: 영문 슬러그 기준 (예: `egg.json`, `green-onion.json`)
- 한 파일 = 한 식재료

## 스키마

| 필드 | 타입 | 설명 |
|------|------|------|
| `id` | string | 영문 슬러그 (파일명과 동일) |
| `이름` | string | 한글 이름 |
| `영문명` | string | 영문 이름 |
| `수량` | number | 보유 수량 |
| `단위` | string | 수량 단위 (개, g, 통, 봉, 단 등) |
| `카테고리` | string | 분류 (채소·육류·유제품·과일·양념 등) |
| `보관방법` | string | 냉장 / 냉동 / 실온 (복합 표기 가능) |
| `보관위치` | string | 구체적 보관 위치·팁 |
| `소비기한_일` | number | 권장 소비기한(일, 대표 보관 기준 근삿값) |
| `알레르기` | string[] | 식품 알레르기 유발 요소(한국 표시 기준). 없으면 `[]` |
| `아이선호도` | number\|null | 3~7세 아이 선호도 1~5점. 조미료 등 해당 없으면 `null` |
| `주요영양소` | string[] | 대표 영양 성분 |
| `대표요리` | string[] | 해당 재료로 자주 만드는 요리 |
| `비고` | string | 보관·조리·유아식 활용 팁 |

> `알레르기`와 `아이선호도`는 어린아이가 있는 가정을 위해 추가한 필드로, 레시피 스킬이 알레르기 회피·아이 입맛 우선순위를 판단하는 데 활용할 수 있습니다.
